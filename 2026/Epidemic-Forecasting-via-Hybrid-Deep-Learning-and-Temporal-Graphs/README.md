## 🎯 Research Problem

Epidemic forecasting becomes unreliable when surveillance data are noisy, incomplete, or affected by underreporting and rapidly changing population behaviour. These issues reduce the stability of conventional statistical and deep-learning forecasting models.

---

## 💡 Proposed Solution

This study proposes a hybrid deep learning framework for robust epidemic forecasting by combining:

* Visibility graph construction
* Temporal graph modelling
* Stochastic noise modelling
* Graph fusion strategies
* Lightweight forecasting models
* Cloud-oriented deployment

The framework transforms epidemic time-series data into uncertainty-aware temporal graph representations to improve robustness under noisy and incomplete surveillance conditions.

---

## ⭐ Key Contributions

* Hybrid epidemic forecasting framework using visibility and temporal graphs
* Stochastic noise modelling using fractional Brownian motion and Lévy-type dynamics
* Graph fusion mechanism for uncertainty-aware representation
* Robust evaluation under noise and missing data
* Improved forecasting performance compared with ARIMA, LSTM, and GCN baselines
* Cloud-oriented deployment for low-latency epidemic monitoring
