# A Model for Predicting the Spread Patterns of Human and Computational Epidemics on Complex Temporal Networks

## Overview

This publication presents a unified framework for predicting the spread of both human epidemics and computational epidemics using complex temporal networks.

The proposed approach integrates temporal network analysis, data-driven temporal discretization, explicit noise modeling, and machine learning techniques to improve prediction accuracy across different epidemic scenarios. The framework enables comparative analysis between biological disease outbreaks and cyber malware propagation within a common analytical model.

## Abstract

This study proposes a unified prediction framework for analyzing the spread of both human epidemics and computational epidemics using complex temporal networks.

The proposed framework combines temporal network modeling, explicit noise representation, optimized temporal discretization, and machine learning techniques to capture dynamic spreading behaviors across heterogeneous networks. Experimental evaluation demonstrates that the proposed approach improves prediction robustness while supporting cross-domain analysis between healthcare and cybersecurity applications.

The proposed framework provides a practical foundation for future intelligent epidemic prediction systems and decision-support tools.
